## Overview

Worked as a key Android developer on a large-scale live-streaming and social platform, delivering features that directly impacted user engagement and revenue growth across multiple international markets.

## Key Contributions

- Developed and maintained real-time features including live video streaming, interactive chat systems, and community engagement tools using Kotlin and Jetpack Compose
- Implemented AI-powered recommendation systems and content personalization to boost user retention
- Built monetization features including in-app purchases and virtual gifting mechanisms
- Contributed to multi-market expansion supporting English, German, French, and Arabic localizations
- Optimized app performance for smooth streaming experiences even on lower-end devices
- Collaborated with cross-functional teams to ship features on tight deadlines in a fast-paced agile environment

## Technologies

Kotlin, Jetpack Compose, Coroutines & Flow, WebSocket, ExoPlayer, Dagger 2/Hilt, Clean Architecture, MVI, Firebase, CI/CD
