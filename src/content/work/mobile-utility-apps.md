## Overview

Designed and built several utility and productivity Android applications from the ground up, focusing on intuitive user interfaces, efficient local data storage, and clean architectural patterns.

## Key Contributions

- Created expense tracking and inventory management applications with full CRUD functionality
- Built modern, responsive UIs using Jetpack Compose and Material 3 design guidelines
- Implemented local data persistence with Room and Realm databases, including data migration strategies
- Integrated REST APIs for data synchronization and third-party service connectivity
- Applied Clean Architecture with MVVM pattern for testable, maintainable codebases
- Delivered polished apps to the Google Play Store with proper release management

## Technologies

Kotlin, Jetpack Compose, Material 3, Room, Realm, Retrofit, MVVM, Dagger 2/Hilt, Coroutines & Flow, Google Play Console
